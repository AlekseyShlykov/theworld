# The World

An interactive strategy long-read game exploring key milestones in human history.

## Overview

This browser-based game is inspired by Jared Diamond's "Guns, Germs & Steel" and lets players experience how geography, environment, and resources shaped human civilization across different regions of the world.

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **HTML5 Canvas** - Map rendering and area overlays
